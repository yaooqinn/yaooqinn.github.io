## 🦊 Apache Kyuubi

**角色：** PMC 主席、副总裁

一个分布式多租户网关，提供数据仓库和湖仓上的 Serverless SQL 服务。我于 2021 年将该项目从网易捐赠至 Apache 孵化器，2022 年毕业成为 ASF 顶级项目。

🔗 [GitHub](https://github.com/apache/kyuubi) · [官网](https://kyuubi.apache.org/)

## ✨ Apache Spark

**角色：** PMC 成员

大规模数据处理的统一分析引擎。我一直在为 Spark SQL 做贡献，专注于 SQL 兼容性、数据类型和配置改进。

🔗 [GitHub](https://github.com/apache/spark) · [官网](https://spark.apache.org/)

## 🌟 Apache Gluten

**角色：** PMC 成员

🔗 [GitHub](https://github.com/apache/incubator-gluten) · [官网](https://gluten.apache.org/)
