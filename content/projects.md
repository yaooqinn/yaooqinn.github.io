## 🦊 Apache Kyuubi

**Role:** PMC Chair, Vice President

A distributed and multi-tenant gateway to provide serverless SQL on data warehouses and lakehouses. I donated the original project from NetEase into the Apache Incubator in 2021, and it graduated as an ASF Top-Level Project in 2022.

🔗 [GitHub](https://github.com/apache/kyuubi) · [Website](https://kyuubi.apache.org/)

---

## ✨ Apache Spark

**Role:** PMC Member

A unified analytics engine for large-scale data processing. I've been contributing to Spark SQL, focusing on SQL compatibility, data types, and configuration improvements.

🔗 [GitHub](https://github.com/apache/spark) · [Website](https://spark.apache.org/)

## 🌟 Apache Gluten

**Role:** PMC Member

A plugin to double SparkSQL's performance by offloading the SQL engine's execution to native engines. Gluten graduated as an ASF Top-Level Project.

🔗 [GitHub](https://github.com/apache/incubator-gluten) · [Website](https://gluten.apache.org/)
