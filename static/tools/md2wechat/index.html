📝 微信公众号 Markdown 排版工具

将 Markdown 文档转换为微信公众号友好的富文本格式 · 支持 Scala / SQL / Python / Java 语法高亮

Markdown 输入 支持拖拽 .md 文件到此处
预览效果 复制后粘贴到公众号编辑器

💡 使用说明

  • 在左侧输入 Markdown 内容，或点击"加载 .md 文件"导入本地文件
  • 点击"复制到公众号"按钮，然后在微信公众号编辑器中 Ctrl+V 粘贴
  • 支持标题、代码块、表格、引用、列表、图片等常用 Markdown 语法
  • 代码块支持 Scala、Java、SQL、Python、Bash、JavaScript 等语言的语法高亮
  • ⚠️ 微信不支持外部图片链接，粘贴后需在公众号编辑器中手动上传图片
